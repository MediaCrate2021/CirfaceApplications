Migration Tool is an internal application operated by Cirface. Access is restricted to authorised Cirface staff only. Unauthorised use is prohibited.

2. Acceptable Use

This tool is provided to facilitate authorised data migration services on behalf of clients. You must:

  • Only use this tool for legitimate, client-approved migration work.
  • Ensure you have the client's consent before accessing their data.
  • Handle all client data in accordance with Cirface's data handling policies and applicable privacy laws.

3. Non-Destructive Operations

This tool is designed to be non-destructive. Source data is never modified or deleted. However, migrations to existing Asana projects will add tasks to that project, which cannot automatically be reversed. Exercise caution when selecting an existing project as a destination.

API tokens and credentials entered into this tool are held only for the duration of your session and are never stored persistently. You are responsible for keeping your credentials secure and for any actions performed using your account.
